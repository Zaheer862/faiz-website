# F.AI.Z Website — V2

> **Faiz Mobile Accessories Ltd** — Phone, Tablet & Computer Repair, Leicester
> 241B Mere Road, Leicester LE5 5GS | 07440 423 053

This is the **V2 development project** for the F.AI.Z website. The goal is to improve the site across design, mobile experience, new features, SEO, and local reach — then publish it on a custom domain.

See [PROJECT.md](PROJECT.md) for the full improvement roadmap and task list.

---

## Quick Start

No server needed. Just open `index.html` in any browser.

---

## Editing Content

### Update phone / address / email
Open `index.html` and search for the values you want to change.

### Add or update products
Edit `products.json`. Each product:
```json
{
  "id": 1,
  "name": "iPhone 15 Pro 256GB",
  "category": "phones",
  "price": 799.99,
  "description": "Latest iPhone, excellent condition.",
  "badge": "Refurbished",
  "image": "img/iphone15.jpg"
}
```
Categories: `phones` · `tablets` · `covers` · `accessories`
